ABOUT ME

Saya merupakan mahasiswa Ilmu Komputer di Universitas Qamarul Huda Badaruddin Bagu yang memiliki ketertarikan pada pengembangan front-end web. Saya menguasai HTML, CSS, dan Bootstrap untuk membuat tampilan website yang responsif dan enak dilihat. Saya senang mengubah desain menjadi website yang siap digunakan serta terus mengembangkan kemampuan melalui latihan dan proyek sederhana.

Ke depannya, saya memiliki tujuan untuk berkembang menjadi seorang web developer fullstack. Saya ingin memperluas kemampuan tidak hanya pada sisi tampilan, tetapi juga pada pengelolaan logika aplikasi dan sistem di sisi server. Dengan terus belajar dan mencoba berbagai teknologi, saya berharap dapat membangun aplikasi web yang utuh, fungsional, dan bermanfaat.
